Installation Guide

1. Install and set up the Modrinth App.

2. Click the plus button on the left-hand sidebar.

3. Select the 'From File' option at the top.

4. Add in the latest version of the modpack. Can be downloaded here.

5. Once it has set itself up, go into it from the 'Library' page and press Play.

6. Enjoy the server! If you need it, you can get the server IP from the Server Information page.
